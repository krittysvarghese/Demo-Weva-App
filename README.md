# Video Maker Flutter Application
### app_demo_weva

This project is the UI for a Video Maker Flutter Application similar to Animoto.

Project includes:
  * Google SignIn & Firebase AUthentication.
  * 3 Tabs
      * Templates Page (data read from json file)
      * Videos Page
      * User Settings Page
